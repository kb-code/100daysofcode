What it is...

Today's exercise is an American Flag layout using CSS grid and flexbox styling. Not because I'm overly patriotic, but because the layout of the flag seemed like a good and achievable grid display exercise.

During this exercise, I constrained myself to only use things I actually remembered how to do without consulting a reference. And since it's Day 1... well, I am a little rusty. :p Given a little more time and creative leeway, I might:

  1. Draw the stars using pure css instead of a png file
  2. Make the size of the flag more flexible for different screens
  3. Attempt some sort of waving effect
  4. Use different message across it. (This was the only song I remembered from singing it in school as a kid!)
  5. Get rid of the cheats. No, you can't see them, but I know they're there!
